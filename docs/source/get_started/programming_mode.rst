Programming Mode
==========================

**Overview**

当你体验过 :ref:`ar_play_mode` 之后，你想要理解每个功能是如何实现的，然后根据自己的想法来修改效果，你可以来仔细阅读这个编程模式。

在编程模式章节中，会将每个功能做成单独的项目，共有18个，在每个项目中都会有对应的代码及配套的原理讲解，你可以边学边体验各个功能控制。

使用的是Arduino编程语言，所以你需要先安装Arduino IDE和相关库之后，再去学习各个项目。

现在让我们开启探索之旅吧！
