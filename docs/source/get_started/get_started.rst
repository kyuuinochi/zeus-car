Get Started
=====================

我们为Zeus Car设定了两种模式，上手即玩模式和编程模式。

* 上手即玩的模式指的是你只需要将Zeus Car开机，然后就能直接用遥控器或者手机APP来控制这个小车，让它实现各种炫酷的功能。因为我们在出厂的时候已经给它预上传了代码。

* 编程模式就是等你体验过Zeus Car的各种炫酷功能之后，你可以单独根据各个项目来了解各个功能的实现原理，然后根据自己的想法来修改，来实现自己想要的功能。
